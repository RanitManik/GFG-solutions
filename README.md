## Folder Structure

The solutions are organized by GeeksforGeeks problem name. Each problem has its own folder containing the solution files in different programming languages.

```
GFG-solutions
│
├── Problem1
│   ├── solution.js
│   ├── README.md
│   └── NOTES.md
│
├── Problem2
│   ├── solution.java
│   ├── solution.py
│   └── README.md
│
└── ...
```
